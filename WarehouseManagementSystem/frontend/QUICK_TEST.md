# 快速测试指南（5分钟）

## 前置条件
- 后端API已启动（`http://localhost:5000`）
- 前端开发服务器已启动（`http://localhost:5173`）
- 有效的用户账号

## 快速测试步骤

### 第1步：测试登录（1分钟）
```
1. 打开 http://localhost:5173
2. 输入用户名和密码
3. 点击"登录"
✅ 预期：成功登录，显示仪表板
```

### 第2步：测试导航（1分钟）
```
1. 点击左侧菜单"储位管理"
✅ 预期：显示储位列表

2. 点击左侧菜单"任务管理"
✅ 预期：显示任务列表

3. 点击左侧菜单"仪表板"
✅ 预期：显示仪表板
```

### 第3步：测试储位管理（2分钟）
```
1. 进入"储位管理"页面
2. 在搜索框输入储位名称，点击"搜索"
✅ 预期：列表过滤显示搜索结果

3. 点击任意储位行
✅ 预期：弹框打开显示储位详情

4. 在弹框中点击"清空物料"
✅ 预期：显示确认对话框，操作成功后关闭弹框

5. 勾选多个储位复选框
✅ 预期：批量操作工具栏出现

6. 点击"批量清空物料"
✅ 预期：显示确认对话框，操作成功
```

### 第4步：测试任务管理（1分钟）
```
1. 进入"任务管理"页面
✅ 预期：显示任务列表

2. 点击"新增任务"按钮
✅ 预期：跳转到任务创建页面

3. 选择源位置和目标位置
✅ 预期：位置选择器正常工作

4. 点击"创建任务"
✅ 预期：创建成功后返回列表页面
```

### 第5步：测试登出（1分钟）
```
1. 点击右上角用户菜单
2. 点击"退出登录"
✅ 预期：重定向到登录页面
```

## 常见问题快速解决

### 登录失败
- 检查用户名和密码是否正确
- 检查后端API是否运行
- 打开浏览器控制台查看错误信息

### 页面空白
- 刷新页面（F5）
- 清除浏览器缓存
- 检查浏览器控制台错误

### 数据不显示
- 检查后端数据库是否有数据
- 检查API是否返回正确的数据
- 打开浏览器Network标签查看API响应

### 操作失败
- 检查浏览器控制台错误
- 检查Network标签中的API响应
- 确保令牌未过期
